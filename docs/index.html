Run Locally with Docker

The fastest way to see the full interactive app on your machine:

git clone https://github.com/thisilyessbr/gestion-soutien-scolaire.git
cd gestion-soutien-scolaire

# Build and start Tomcat + MySQL
docker-compose up --build

# Open http://localhost:8080
# Login with admin / admin

Requires Docker Desktop to be installed and running.

Key Features

  • Role-based dashboards for Admin, Professor, and Student
  • User, course, classroom, and session management
  • Attendance tracking and tuition payment records
  • Announcements and PDF report generation
  • Responsive JSP-based UI with custom CSS
